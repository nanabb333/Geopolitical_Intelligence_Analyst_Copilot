# Roadmap

## V0.1: Publication-Ready Deterministic MVP

Goal: make the local historical analog workflow credible, reproducible, and clear for GitHub publication.

Included:

- local CSV inputs only
- deterministic event classification
- weighted historical analog retrieval
- historical market reaction comparison
- deterministic scenario pathways
- Markdown executive brief generation
- orchestration CLI
- documentation
- examples
- basic tests

Not included:

- live news retrieval
- external APIs
- LLM calls
- forecasts
- probability estimates
- trading or investment recommendations

## V0.2: Transparent Similarity Scoring And Retrieval Diagnostics

Goal: improve analyst trust in the retrieval layer.

Planned:

- richer score diagnostics
- explicit matched and missing dimensions
- analog quality flags
- classification uncertainty notes
- stronger sector and mechanism matching
- retrieval test fixtures

## V0.3: Structured Run Artifacts

Goal: make scenario runs auditable and easier to connect to future product interfaces.

Planned:

- scenario IDs
- saved run folders
- `input.json`
- `classification.json`
- `analogs.json`
- `market_reactions.json`
- `pathways.json`
- `brief.md`

## V1.0: Scenario Intelligence Workflow

Goal: deliver a complete research-to-intelligence workflow suitable for portfolio demonstration and analyst review.

Planned:

- stable CLI
- structured artifacts
- auditable methodology
- strong retrieval diagnostics
- curated examples
- test coverage around core workflow
- dashboard-ready JSON outputs

V1.0 should remain historically grounded and decision-support focused.
